# 리와인드 컨버터 (Rewind Converter)

인스타그램 **공식 데이터 내보내기 zip**을 브라우저 안에서만 분석해, 토스 미니앱 「리와인드」에 붙여넣을 수 있는 **리와인드 코드**로 변환하는 정적 웹 페이지입니다.

👉 사용하기: https://sangho24.github.io/rewind-converter/

## 프라이버시 — 왜 소스를 공개하나요

이 도구의 핵심 약속은 **"데이터가 브라우저 밖으로 나가지 않는다"** 입니다. 말로 하는 약속이 아니라 코드로 검증할 수 있도록 전체 소스를 공개합니다.

- 네트워크 전송 코드가 없습니다 — `fetch` / `XMLHttpRequest` / `sendBeacon` / `WebSocket` 을 저장소 전체에서 검색해 보세요 (파싱용 정규식 문자열 외 0건)
- zip 해제·파싱·집계·인코딩 전 과정이 웹워커(`worker.js`) 안에서 로컬로 실행됩니다
- 아이디·비밀번호를 요구하지 않습니다 — 인스타그램 계정 접근·자동화가 일절 없습니다
- CSP(Content-Security-Policy)로 외부 리소스 로드를 차단합니다
- DM은 **메시지 본문을 읽지 않으며**, 발신자·시간 메타데이터만 집계합니다

## 사용 방법

1. 인스타그램 앱 → 설정 → 개인정보 센터 → 내 정보 다운로드 → **HTML 형식**으로 요청 (보통 몇 시간, 길면 며칠 소요)
2. 메일로 받은 zip 파일(여러 개면 전부)을 이 페이지에 끌어다 놓기
3. 분석 완료 후 **"리와인드 코드 복사"** → 토스 미니앱 「리와인드」에 붙여넣기

지원: 한국어·영어 내보내기, 분할 zip. 코드에는 회원님과 팔로워들의 계정명이 포함되니 **다른 사람에게 공유하지 마세요.**

## 구조

```
index.html        UI (업로드·진행률·결과·코드 복사)
worker.js         웹워커 — zip 스트리밍 해제, HTML 파싱, 집계, RW1 인코딩
lib/fflate.min.js zip 해제 라이브러리 (로컬 번들)
```
